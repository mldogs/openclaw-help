# OpenClaw Help

Репозиторий с полезными материалами по **OpenClaw**: настройка, интеграции, плагины, эксплуатация, отладка и прочие вещи, на которых обычно теряется время.

Идея простая: складывать сюда не абстрактную теорию, а короткие и полезные документы, которые реально помогают быстрее что-то поднять, починить или понять.

## Что это за репозиторий

Это не репозиторий только про Langfuse.

Это репозиторий про **полезности для OpenClaw** в широком смысле:
- интеграции
- troubleshooting
- документация по практическим сценариям
- заметки по плагинам и runtime hooks
- шаблоны и рабочие подходы

Langfuse сейчас просто первая оформленная тема, потому что она уже была добита до рабочего состояния и её имело смысл сразу задокументировать.

## Принципы этого репозитория

- **Практичность важнее полноты.** Лучше один рабочий документ, чем десять пустых обещаний.
- **Факты важнее фантазий.** Если что-то не проверено, оно не должно выглядеть как проверенное.
- **Безопасность важнее удобства.** Без токенов, ключей и случайных сливов.
- **Повторяемость важнее шаманства.** Если шаг нельзя воспроизвести, он описан плохо.

## Для кого

Для тех, кто:
- уже использует OpenClaw
- встраивает вокруг него свои плагины и интеграции
- хочет быстрее разбираться в реальных operational-вопросах
- не хочет заново наступать на одни и те же грабли

## Важное ограничение

В репозитории не должно быть:
- секретов
- приватных токенов
- конфигов с чувствительными значениями
- лишнего внутреннего контекста, который не нужен для понимания
